## Techjob
Basic job site using &lt;flask>  
## INSTALL
clone this repository, change directory to this clone folder and run following commands  
1. ```virtualenv -p python venv```  
2. ```pip install -r requirements.txt```  
3. ```python crawl.py```  
4. ```python app.py```
